## KEY DISCOVERIES

### The Bloom Issue (MOST IMPORTANT!)
- UnrealBloomPass with threshold=0, strength=1.5, radius=0.5
- Must use EffectComposer
- Without this, game looks flat!

### Ship Geometry
- 56 decorative hull boxes
- Multiple toroid rings (12.15 diameter critical!)
- Nacelle pods with 5-6 sub-geometries
- 3 sensor lights with glow materials
- Multiple emissive materials pulsing at different speeds

### Starfield (Three Layers!)
1. Main: 5000 stars, r=500-2500, size=2, opacity=0.8
2. Bright: 500 stars, r=100-500, size=4, opacity=1.0
3. Dust: 10000, random -2000:+2000, size=0.5, opacity=0.5

Each rotates independently for parallax effect.

### Controls
Uses OrbitControls - camera orbits ship (not flight controls!)

### Shield
Sphere with fresnel + sine ripple shader, only shows when baseGlow>0

### Warp
Shift key: speed 10x, starfield Z-stretch 1→10/20

## CRITICAL SETTINGS TO USE

Camera: FOV 75, near 0.1, far 10000
Bloom: threshold=0, strength=1.5, radius=0.5
Toroid: TorusGeometry(12.15, 0.24, 18, 120)
Colors: Tan (#D6BD99), Cyan (#00FFFF), Magenta (#FF8EFC)
